# FrankenSite
A YSWS for clubs to design their own version of this webpage through submitting pull requests.

Here's how to do this with your club:

1. Clone this repository.
2. Show it to the club. Have them come up with ideas and research how to implement.
3. Code their additions and modifications.
4. Submit pull requests to your club's cloned repository
5. Participants submit this form describing their contributions.
6. Your club's website will then be live in the FrankenSite rotation!

There's only one rule in FrankenSite....Ok, maybe two:

1. The website must remain functional. People must be able to understand the program and participate. Other than that, anything goes.
2. Keep it friendly for all audiences. This will be displayed rotated live on Hack Club's FrankenSite an example. We don't want to freak out any wandering teacher eyes.

What could be added:
1. More CSS.
2. Links or logos for your club.
3. A background of memes.
4. Add hidden links for other websites. It's a chance to Rick Roll people!
5. Change the cursor to something random.
6. Make something trail behind the cursor.
7. Anything!
